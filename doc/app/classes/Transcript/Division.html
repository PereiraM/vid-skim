method_missing(method_id, *args)

Builds a dynamic finder (unique_entries_by_attribute where attribute is an Entry attribute) so that filters returns unique entries you can do things like: 

  >> entries = [{'title'=>'Hamm', 'range'=>["00:00:00", "00:00:00"]},
              {'title'=> 'Clove', 'range'=>["00:00:00", "00:00:00"]},
              {'title'=>'Hamm', 'range'=>["00:00:00", "00:00:00"]}]
  >> d = Transcript::Division.new('Endgame')
  >> d.entries = entries
  >> uniq = d.unique_entries_by_title
  >> uniq.each {|u| p u.title }
  "Hamm"
  "Clove"
